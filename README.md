##文档说明
  + api 文档 用来做数据请求接口的事情

  + assets用来放置静态文件

  + component 用来放置公共的组件

  + pages 用来放置 所有的主要页面，依次为：

    - 首页 home
    - 分类页 classify
    - 发现页 find
    - 购物车页 shopcar
    - 我的页 my
    - 404 ye notFound
    - ** index.js 文件用来配置loading效果
  + routes 文件夹用来放置所有的路由表的配置
    **规范**
    一级路由为：
    ```
    "/home"
    "/classify"
    "/find"
    "shopcar"
    "my"
    "404"
    ```


  + store 用来 存放所有的数据和操作方法
